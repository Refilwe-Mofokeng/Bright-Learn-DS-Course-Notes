# Bright Learn Data Science Course Notes

## Overview

This repository contains my notes, completed as part of the **Bright Learn Data Science Course**.

The purpose of this repository is to document my learning journey, consolidate key concepts, and create a reference resource for future Data Science projects and professional development.

Topics covered include:

* Data Science Fundamentals
* Databases and Data Warehousing
* SQL Fundamentals
* SQL Filtering and Logical Operators
* Aggregate Functions
* Grouping and Aggregations
* CASE Statements
* SQL Joins
* NULL Functions
* Date Functions
* BigQuery
* Databricks
* Data Cleaning and Transformation
* Data Visualisation
* Business Analytics Fundamentals
* KPI Identification and Measurement
* Project Planning and Documentation

## Course Objectives

Through this course, I aimed to:

* Build a strong foundation in Data Science concepts
